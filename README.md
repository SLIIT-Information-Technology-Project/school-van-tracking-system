# school-van-tracking-system (IT2150 – IT Project)

This system uses modern IT technologies such as GPS, real-time tracking, and mobile/web applications to provide live location updates, timely notifications, and attendance tracking. It reduces uncertainty for parents, improves route management for schools, and enhances overall transport safety.

- Firebase / MongoDB
- Google Maps API (GPS Tracking)

Main Features:
- Live van location tracking
- Student pickup/drop attendance
- Parent notifications
- Route & schedule management
- Driver monitoring
